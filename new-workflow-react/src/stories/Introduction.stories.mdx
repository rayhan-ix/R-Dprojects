# Welcome to Storybook of New Workflow App

Storybook helps you build UI components in isolation from your app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during development, testing, or QA.

Browse stories by navigating to them in the sidebar.
Storybook recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.
